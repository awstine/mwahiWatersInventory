## âœ¨ Features

- **Fully Responsive Design**: A mobile-first approach ensures the site looks great on all devices, from phones to desktops.
- **Single-Page Layout**: Smooth-scrolling navigation for a seamless user experience.
- **Dynamic Hero Section**: Features a subtle, auto-playing background image slider.
- **Interactive Contact Form**: A pop-up modal allows users to send messages directly. Form submissions are handled by [Web3Forms](https://web3forms.com/) and sent to an admin email.
- **Scroll-Triggered Animations**: Engaging fade-in and slide-up animations as the user scrolls through sections.
- **"Back to Top" Button**: Appears on scroll for easy navigation back to the start of the page.
- **Accessibility Conscious**: Includes `aria-label` attributes and respects `prefers-reduced-motion` settings.
